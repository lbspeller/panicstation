# What to do if… your home Wi-Fi name or password appears changed and your devices disconnect (USA)

## Short answer
Treat this as possible router access until you have secured it. Use a wired connection or your ISP/router app if you can, reset the router if needed, then change the router admin password and Wi-Fi password before reconnecting everything.

## Do not do these things
- Don’t log into banking, email, work, or password-manager accounts on that Wi-Fi until the router is secured.
- Don’t only change the Wi-Fi password and stop there; the router admin login also needs securing.
- Don’t reconnect every phone, laptop, camera, speaker, and smart device before you have checked the router settings.
- Don’t enable remote administration, WPS, or UPnP just to get devices back online quickly.
- Don’t call a random “router support” number from a pop-up, search ad, text, or email.
- Don’t reuse the old Wi-Fi password or an easy password.

## What to do now
1. **Make sure you are working on your own router.**  
   Check the router label, your ISP app, or your router app for the expected network name and setup route. If you can, connect one computer to the router with an Ethernet cable while you fix the settings.

2. **Do one brief restart.**  
   Unplug the router from power, wait briefly, and plug it back in. If the Wi-Fi name or password still looks wrong after it restarts, treat the router settings as changed.

3. **Try to enter the router admin page or app.**  
   Use the admin address, app, or setup method from the router label, ISP instructions, or manufacturer instructions. Do not use links from pop-ups or unexpected messages.

4. **If you cannot trust or access the settings, factory reset the router.**  
   Follow the reset instructions for your exact router or ISP gateway. A reset usually removes unknown settings, but it may also remove custom Wi-Fi names, port forwarding, parental controls, guest networks, and mesh settings.

5. **Change the router admin password first.**  
   Set a strong, unique router admin password. If the router or ISP account uses an app or online portal, change that account password too and enable two-factor authentication if it is offered.

6. **Update the router.**  
   Check for firmware or software updates in the router app, admin page, ISP app, or manufacturer page. Turn on automatic updates if the router offers them. If the router no longer receives updates, contact the ISP or manufacturer about replacement.

7. **Check the settings that commonly matter after a router change.**  
   In the router settings, review and correct:
   - DNS settings you did not choose
   - port forwarding rules you did not create
   - remote management or remote administration
   - WPS
   - UPnP
   - guest network name and password
   - router firewall setting, if shown

8. **Recreate the Wi-Fi with safer settings.**  
   Use WPA3 Personal if available, or WPA2 Personal if WPA3 is not available. Do not use WEP or WPA. Set a long, unique Wi-Fi password and use a network name that does not include your name, address, phone number, or router brand.

9. **Reconnect devices slowly.**  
   Start with one phone or computer. Then check the router’s connected-device list. Device names can be unclear, so identify household devices before blocking anything; block or remove devices that remain unknown.

10. **Secure the accounts that could have changed the router.**  
   Change the password for your ISP account, router app account, and the email account used for password resets. Update the phone or computer you used to manage the router, and run reputable security checks where available.

11. **Report only if there is fraud, cybercrime, or identity risk.**  
   If money was lost or this appears connected to an online crime, use the FBI IC3. If personal information was misused, use IdentityTheft.gov. If it was part of a scam or fraud attempt, use ReportFraud.ftc.gov.

## What can wait
- You do not need to know exactly how it happened right now.
- You do not need to rebuild your whole smart home today.
- You do not need to replace every device before the router is secured.
- You can decide later whether to create separate guest or smart-device networks.
- You can do a deeper password-manager cleanup after the router and key accounts are stable.

## Important reassurance
A changed Wi-Fi name or password can feel invasive, but the first steps are practical and limited: regain router control, reset unknown settings, update it, and use new passwords. You are not expected to solve the whole cause at once.

## Scope note
These are first steps to regain control of a home router and reduce immediate risk. Mesh systems, custom firewalls, business equipment, work VPN devices, or ISP-managed gateways may need model-specific help after the immediate situation is stable.

## Important note
This is general information, not professional cybersecurity, legal, financial, or technical advice. A changed Wi-Fi name or password does not prove hacking; outages, ISP changes, resets, hardware faults, and setup mistakes can also cause similar symptoms.
