# What to do if… you accidentally permanently delete an important folder and need to recover it today (USA)

## Short answer
If the folder was on a local or external drive, stop using that drive and minimize activity on the computer. Check built-in restore options first, and avoid further changes to a cloud-synced folder while recovery is underway.

## Do not do these things
- Don’t install recovery software on the affected computer or drive before checking built-in restore options.
- Don’t save new files, download updates, or run cleanup tools on an affected local or external drive.
- Don’t empty any Recycle Bin, Trash, or cloud trash folder.
- Don’t make repeated experimental changes. Follow built-in restore steps first, or use steps recommended by your IT team or a recovery specialist.
- Don’t assume the folder is unrecoverable until you have checked the relevant restore paths.

## What to do now
1. **Identify where the folder was stored.**  
   Check whether it was on a Windows computer, a Mac, an external drive, a network share, OneDrive, SharePoint, Google Drive, or iCloud Drive. For Microsoft Teams, note whether it was a channel file or a chat file: channel uploads are usually stored in SharePoint, while chat files are usually stored in OneDrive for Business. Write down the folder path and the approximate deletion time.

2. **Protect a local or external drive from further writes.**  
   Stop downloads, software updates, file saves, cleanup tools, and recovery-tool installations. Use another device if practical. If you must use the computer, keep activity to a minimum and save any new files to a different drive.

3. **Check the computer’s built-in deleted-items location.**
   - **Windows:** Open Recycle Bin and look for the folder.
   - **Mac:** Open Trash and look for the folder.
   - If you find it, restore it before trying other steps.

4. **Check the relevant cloud restore path.**
   - **OneDrive:** Sign in on the OneDrive website and open Recycle bin. For a personal account, deleted items are normally removed from the recycle bin after 30 days. For a work or school account, the usual period is 93 days unless an administrator changed the setting. If many files were affected, Microsoft 365 subscribers can use **Restore your OneDrive** to undo file and folder actions from within the last 30 days. Review the selected activity before restoring because this can undo other changes after the chosen restore point.
   - **SharePoint or Teams channel files:** Check the site Recycle Bin. If the item is not there, ask an owner or administrator to check the second-stage Recycle Bin.
   - **Teams chat files:** Check the OneDrive for Business Recycle bin for the account that uploaded the files.
   - **Google Drive:** Check Trash. Items normally remain there for 30 days before they are permanently deleted. For a Google Workspace account, ask an administrator promptly if Trash was emptied or the item was automatically removed after 30 days. Administrator recovery may still be available within 25 days after permanent deletion. Give the administrator the approximate deletion date so they can use the narrowest practical date range.
   - **iCloud Drive:** On iCloud.com, open iCloud Drive and check **Recently Deleted**. Apple says deleted files can be recovered within the last 30 days unless they were permanently removed.

5. **Check an existing backup or previous version.**
   - **Windows:** In File Explorer, go to the folder that used to contain the deleted folder. Right-click it and select **Restore previous versions**. Available versions depend on whether File History or another backup was already configured. Preview the version first and, when the option is available, restore it to a different location before replacing current files.
   - **Mac:** If Time Machine was already configured, browse its backups and restore the folder.
   - **Network share:** Ask your IT team whether a snapshot or backup restore is available.

6. **For a work or school folder, contact IT promptly.**  
   Give them the folder path, approximate deletion time, device name, storage service, and any steps already taken. Ask them to check recycle bins, version history, snapshots, and backups.

7. **Look for a safe copy needed for today.**  
   Without writing to the affected local or external drive, check shared drives, email attachments, collaboration tools, exported ZIP files, and copies held by coworkers or classmates.

8. **Stop experimenting if the folder is still missing and the contents are critical.**  
   For a local or external drive, ask a reputable data-recovery specialist before making more changes. Additional writes to the affected drive may overwrite data that could otherwise be recoverable.

## What can wait
- You don’t need to choose new backup software today.
- You don’t need to pay for a recovery tool before checking built-in restore options, existing backups, and any IT-admin restore path.
- You don’t need to clean up, reorganize, or optimize the affected drive.
- You don’t need to decide on a long-term storage plan until the immediate recovery attempt is complete.

## Important reassurance
A permanent-looking deletion can still have a built-in restore path, especially when a recycle bin, cloud trash folder, previous version, or backup exists. Taking a few careful steps is more useful than trying many fixes at once.

## Scope note
These are first steps only. A workplace restore, a complex sync issue, or recovery from a local drive may need help from an IT team or a reputable data-recovery specialist.

## Important note
This is general information, not professional IT, legal, forensic, or data-recovery advice. Available restore options depend on the device, service, account type, configuration, and how much the affected drive has been used since deletion.
