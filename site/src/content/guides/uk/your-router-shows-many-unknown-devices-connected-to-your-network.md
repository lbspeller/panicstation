# What to do if… your router shows many unknown devices connected to your network (UK)

## Short answer
Move any sensitive activity to mobile data for now, then change the router admin password and Wi-Fi password. After that, use WPA2 or WPA3 security, turn off WPS and remote administration if they are on, and update the router software if an update is available.

## Do not do these things
- Don’t assume the device names are accurate; routers often show phones, printers, TVs, smart plugs, and speakers as “unknown” or “generic”.
- Don’t treat blocking one device as the main fix; it may miss other devices and does not fix how access happened.
- Don’t do banking, email password changes, or other sensitive logins on this Wi-Fi until you have secured it; use mobile data if you can.
- Don’t share screenshots of your router admin page publicly, because they can expose network names, device details, and settings.
- Don’t factory-reset in a panic if you can still control the router settings; do the safer password and security changes first.

## What to do now
1. **Move important activity away from this Wi-Fi for the next few minutes.**  
   On your phone or laptop, switch to mobile data if you need to use banking, email, shopping, work accounts, or password reset pages.

2. **Keep access to the router settings while you work.**  
   If you know how to pause the internet connection without losing access to the router admin page, you can do that briefly. If you are not sure, leave the router on and continue with the password changes.

3. **Log into the router and change the router admin password first.**  
   This is the password used to change router settings, not the Wi-Fi password. Use a long, unique password that you do not use anywhere else.

4. **Change the Wi-Fi password.**  
   Use a long, unpredictable password. Save the change, then reconnect only your own devices with the new Wi-Fi password.

5. **Set the Wi-Fi security mode to WPA2 or WPA3.**  
   In the wireless or security settings, choose WPA2-Personal, WPA3-Personal, or WPA2/WPA3 mixed mode if needed for older devices. Do not use WEP or an open network.

6. **Turn off WPS.**  
   Look for “WPS”, “Wi-Fi Protected Setup”, “push button setup”, or “PIN setup”. Turn it off if the router allows this.

7. **Turn off remote administration if it is on.**  
   Look for “Remote Management”, “Remote Admin”, “Admin over WAN”, or similar. If you do not specifically use it, turn it off.

8. **Turn off UPnP unless you know you need it.**  
   Look for “UPnP” or “Universal Plug and Play”. If a game, voice service, or device stops working later, you can review this setting once the network is stable.

9. **Update the router software if the option is available.**  
   Look for firmware, software update, or automatic update settings. If your ISP supplied the router and you cannot find this, check the ISP’s help page or ask the ISP how updates are handled.

10. **Work through your own devices calmly.**  
   Check phones, laptops, tablets, smart TVs, streaming sticks, game consoles, printers, smart speakers, plugs, cameras, and hubs. If you cannot identify a device, keep it locked out by not sharing the new Wi-Fi password.

11. **If unknown devices still appear after these changes, do a controlled factory reset.**  
   Reset the router only when you have time to set it up again. After the reset, set a new admin password, set a new Wi-Fi password, use WPA2 or WPA3, turn WPS off, turn remote administration off, update the router software, and ask your ISP for help if it supplied the router.

12. **If you suspect fraud, account access, or a crime, handle that separately.**  
   If money was taken or a payment account may be affected, contact your bank or provider first using a trusted route. For cyber crime or fraud in England, Wales, or Northern Ireland, use Report Fraud. If you are in Scotland, contact Police Scotland. Use 999 if there is immediate danger or a crime is in progress, and 101 for non-emergency police contact.

## What can wait
- You do not need to identify exactly who every unknown device belongs to right now.
- You do not need to hide the network name, buy a new router, or set up a guest network immediately.
- You can rename devices, tidy the device list, set parental controls, and plan longer-term network changes later.

## Important reassurance
A router device list can look more alarming than it is. Many ordinary home devices use unclear names, and changing the admin password and Wi-Fi password forces a fresh start for anything trying to connect.

## Scope note
These are first steps to stabilise your home network and reduce the chance of further access. If devices keep returning after a reset, or if accounts or money may be affected, you may need help from your ISP, bank, police reporting route, or a trusted technical support provider.

## Important note
This is general information, not legal, financial, technical, or professional advice. If anyone is in immediate danger or a crime is happening now, call 999.
