# What to do if… a USB drive suddenly appears empty or asks to be formatted (UK)

## Short answer
Stop using the USB drive immediately. Do not format, initialise, erase, save to it, or run repair tools until you have copied what you can, made a recovery image, or accepted the risk of changes.

## Do not do these things
- Don’t click **Format disk**, **Initialise**, **Erase**, **Repair**, or similar prompts if you want the old files back.
- Don’t save, download, install, or recover files onto the same USB drive.
- Don’t keep unplugging and replugging it if it disconnects, gets unusually warm, shows 0 bytes, or behaves erratically.
- Don’t run disk repair tools that may write changes to the USB unless you have copied recoverable files, made an image, or accepted the risk.
- Don’t plug an unknown USB drive into a work, school, client, or sensitive computer.

## What to do now
1. **Pause and protect the drive.**  
   Safely eject the USB drive if your computer allows it. Put it aside for a moment. This reduces accidental writes and repeated mount attempts.

2. **Try one simple hardware check.**  
   Use a different USB port on the same computer, avoiding hubs. If it is an external drive or uses an adapter, try a different cable or adapter. If you have a non-sensitive spare computer, try that once.

3. **If it opens anywhere, copy the most important files first.**  
   Copy the files you would be most upset to lose to the computer or to a different drive. Do not reorganise the USB, rename files on it, or copy anything back onto it.

4. **On Windows, check what the computer sees without fixing it.**  
   Open **Disk Management** and find the USB drive by size. If it shows as **RAW**, **Unallocated**, **Unknown**, **Not initialised**, or the right size but empty, stop before formatting or initialising it.

5. **On macOS, check what Disk Utility sees without erasing.**  
   Open **Disk Utility**, choose **View > Show All Devices**, and select the physical USB device as well as any volume underneath it. Note the size and whether it mounts. **First Aid** is Apple’s built-in repair option, but it can make changes, so pause if the files are irreplaceable.

6. **For valuable files, switch from fixing to recovery.**  
   Ask a reputable data recovery specialist, or use a suitable recovery tool, about making an image or clone by reading from the USB and saving the result to a separate drive. The destination must be a different drive with enough space.

7. **Escalate early if the device may be failing.**  
   Use a reputable data recovery service if the USB repeatedly disconnects, shows 0 bytes, gets unusually warm, makes unusual noises, is physically damaged, or contains irreplaceable files. Tell them you have not formatted it and want a read-only assessment where possible.

8. **If it contains work, school, client, or confidential personal data, report it internally.**  
   Tell your organisation’s IT, security, manager, or data protection contact promptly. Follow their process and avoid testing the USB on more organisational devices.

## What can wait
- You do not need to decide today whether the USB drive is dead.
- You do not need to buy replacement storage right now.
- You do not need to run repair commands immediately.
- You do not need to reorganise all your backups today.
- You do not need to explain exactly what caused the problem before preserving the drive.

## Important reassurance
A USB drive can appear empty or unformatted because the computer cannot read the file system, not necessarily because every file has vanished. It may still be possible to recover files, and the main thing you can do now is avoid overwriting or changing the drive.

## Scope note
These are first steps to avoid making data loss worse and to buy time. Later decisions, especially for valuable, sensitive, or work-related data, may need help from an IT professional, a data recovery specialist, or your organisation’s data protection contact.

## Important note
This is general information, not professional IT, forensic, legal, data protection, or security advice. Follow your organisation’s policies where they apply.
