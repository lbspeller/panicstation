# What to do if… you accidentally permanently delete an important folder and need to recover it today (UK)

## Short answer
Stop anything that may write new data to the affected drive or sync location. Then check the fastest reversible places first: cloud recycle bins, local Recycle Bin or Trash, version history, and backups.

## Do not do these things
- Don’t install recovery software onto the same drive the folder was on; it can overwrite recoverable data.
- Don’t keep saving replacement files into the same folder path; use a different drive, USB stick, or cloud location for temporary work.
- Don’t run cleanup tools, optimiser tools, free-space wipe tools, or disk repair utilities before you have checked restore options.
- Don’t keep restarting the affected device or reconnecting the affected external drive unless you need to do so for a specific restore step.
- Don’t assume “permanently deleted” means “impossible” until you have checked cloud recycle bins, version history, backups, and admin restore options.

## What to do now
1. **Pause activity on the affected location.**  
   Stop downloads, installs, large copies, updates, and apps that autosave to the affected drive. If the folder was inside OneDrive, Google Drive, iCloud Drive, SharePoint, or Teams, use a clear pause-sync option if the app offers one. If you are not sure, stop using that synced folder on other devices while you check the web restore tools.

2. **Identify where the folder lived.**  
   Work out whether it was on a Windows or Mac local drive, an external USB drive, a network drive, OneDrive, Google Drive, iCloud Drive, SharePoint, or Teams. Write down the exact folder name, the parent folder, and the approximate time it was deleted.

3. **Check the simplest restore place first.**
   - On Windows, open Recycle Bin and search for the folder name.
   - On Mac, open Trash and search for the folder name.
   - In OneDrive, open the OneDrive website and check Recycle bin.
   - In Google Drive, open Drive on the web and check Trash.
   - In iCloud Drive, check Recently Deleted on iCloud.com.
   - In SharePoint or Teams, open the document library and check Recycle bin.

4. **Use the web version for cloud folders.**  
   If the folder was cloud-synced, check the web recycle bin for that service before trying recovery software. For OneDrive, the “Restore your OneDrive” feature may help roll back recent file and folder actions if you have Microsoft 365 or your work or school account allows it. For Google Workspace, an admin may be able to recover Drive items for a limited time after Trash is emptied, if they have the right privileges.

5. **Check previous versions and backups.**
   - On Windows, open File Explorer, go to the folder that used to contain the missing folder, right-click it, and check **Restore previous versions**.
   - On Mac, use Time Machine if it was set up, go back to a time when the folder existed, and restore the folder.
   - For OneDrive or SharePoint files, check version history for important files that still exist or were restored.
   - For network drives or work shares, ask IT whether snapshots or backups exist for the exact path.

6. **If this is for work or school, contact IT now.**  
   Give the service desk the exact folder path, approximate deletion time, device name, storage location, whether sync was on, and whether you paused it. Ask specifically for a Recycle Bin restore, second-stage Recycle Bin restore, version history restore, snapshot restore, or backup restore.

7. **Make a safe today-only workaround.**  
   Ask collaborators to resend or re-share the folder. Check email attachments, chat uploads, project portals, exports, ZIP files, and old copies. Save any temporary copies somewhere different from the affected drive or sync folder.

8. **If the folder is very valuable and no built-in restore works, stop and preserve it.**  
   Avoid random recovery tools. For a local or external drive, the safer next step is usually a careful recovery approach that does not write to the affected drive, or help from a trusted IT or data recovery professional.

9. **If the folder contains personal data for an organisation, flag it promptly.**  
   Tell your manager, IT team, or data protection lead that a folder containing personal data may have been deleted or lost. Do not decide the legal impact yourself; give the facts so the organisation can assess what records, containment, or ICO steps may be needed.

## What can wait
- You do not need to choose a new backup system today.
- You do not need to buy recovery software before checking recycle bins, version history, backups, and admin restore options.
- You do not need to tidy, defragment, optimise, or make space on the affected drive.
- You do not need to recreate the whole folder before checking whether someone else, a cloud service, or IT can restore it.

## Important reassurance
This happens to careful people, especially with sync tools, shared folders, and keyboard shortcuts. The most useful first move is to reduce further changes and check the built-in restore paths in a calm order.

## Scope note
These are first steps to stabilise the situation and improve the chance of getting the folder back today. Later decisions, such as data recovery, incident handling, or backup changes, may need specialist help.

## Important note
This is general information, not professional IT, legal, forensic, financial, medical, or therapeutic advice. Recovery depends on the device, storage type, account settings, backups, permissions, and how much new data has been written since deletion.
